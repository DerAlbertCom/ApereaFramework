

1. Change your Global.asax.cs Application_Start to
    public class MvcApplication : System.Web.HttpApplication
    {
        protected void Application_Start()
        {
            ApplicationStart.Initialize();
        }

        protected void Application_End()
        {
            ApplicationStart.End();
        }
    }
		 
2. remove the "normal" MVC Initialization (you find the new intialization in the ApereaStart.Initialize)

3. Change you existing Controller to inherit from ApereaBaseController instead of controller

4. Add @RenderSection("scripts", required:false)  to you _Layout.cshtml at the end of <body />
